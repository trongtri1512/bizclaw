⚡ Bắt đầu
Chạy trong 3 phút
Tự cài đặt miễn phí trên thiết bị của bạn. Không cần kiến thức kỹ thuật cao.
01

Clone & Build

Clone repo và build với Cargo

git clone github.com/nguyenduchoai/bizclaw
cd bizclaw && cargo build --release
02

Cấu hình

Wizard tương tác — chọn provider, nhập API key

./bizclaw init
# → Chọn AI provider + API key
03

Chạy!

Dashboard tại localhost:3579

./bizclaw serve
# → http://localhost:3579
04

Hoặc Docker

1 lệnh là xong

docker-compose up -d
# → Tất cả sẵn sàng!
