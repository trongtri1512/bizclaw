🧠

Brain Engine

Chạy mô hình LLaMA ngay trên thiết bị với Flash Attention, FP16 KV Cache (giảm 50% RAM), Grammar-Constrained JSON cho tool calling chính xác.

🔌

15 AI Providers

OpenAI, Anthropic, Gemini, DeepSeek, Groq, OpenRouter, Together, MiniMax, xAI (Grok), Mistral, Ollama, llama.cpp, Brain Engine, CLIProxy, vLLM. Hỗ trợ custom endpoint.

🏢

Multi-Tenant Platform

Admin dashboard, quản lý tenants, JWT auth, mã pairing 6 số, audit log. Mỗi tenant có port riêng, subdomain riêng.

💬

🇻🇳 Zalo OA Bot + Zalo Cá nhân

Dual-mode Zalo: Zalo OA Bot (API chính thức bot.zapps.me — sendMessage, webhook, nhóm chat) + Zalo cá nhân (zca-js v2 protocol, dynamic service URLs). Kết hợp Telegram, Discord, CLI, Webhook. AI agent nói chuyện trên Zalo như người thật.

🔒

Bảo mật

Command allowlist, JWT + bcrypt, AES-256 encryption, HMAC-SHA256 signing, tenant isolation, audit logging.

🔗

MCP Servers — Mở rộng vô hạn

Kết nối hàng trăm MCP servers: GitHub, Postgres, Brave Search, Puppeteer, Memory... Thêm/xoá/bật/tắt ngay trên Dashboard. Tương thích chuẩn MCP v2024-11-05.

🔥

Chạy trên 512MB RAM

Hoạt động độc lập trên BẤT KỲ phần cứng nào có 512MB RAM — từ Raspberry Pi đến VPS mini. LLM sẵn có, không cần cloud. SIMD auto-dispatch (ARM NEON, x86 AVX2).

🎨

Onboarding Wizard + AI Personalization

Lần đầu mở Dashboard → Wizard hỏi bạn là ai → AI tự tạo SOUL.md, IDENTITY.md, USER.md. Agent có tính cách riêng, hiểu bạn ngay từ đầu.

🐳

Docker + One-Click Install

docker-compose up → chạy ngay. Hoặc curl install.sh → cài trên VPS/Pi trong 1 phút. Không cần kiến thức kỹ thuật cao.

📦

Agent Gallery — 50+ Mẫu cho Doanh nghiệp

14 danh mục doanh nghiệp: HR, Sales, Marketing, Finance, Legal, IT, Operations, Management... Cài agent chuyên biệt chỉ 1 click + upload file .md mở rộng kiến thức. Tự tạo Gallery riêng.

💰

Tiết kiệm Token — Mỗi Agent chọn Provider riêng

Translator → Ollama/qwen2.5 (miễn phí, local). Full-Stack Dev → Anthropic/Claude (mạnh). Social Media → Gemini/flash (nhanh, rẻ). Tiết kiệm 60-80% chi phí API.

🔀

Multi-Agent Orchestration

Agents tự phối hợp: Delegation (sync/async), Handoff, Evaluate Loop, Quality Gates, Permission Links, Team Task Board, LLM Tracing. 9 API endpoints sẵn sàng.

🐘

PostgreSQL Backend

bizclaw-db crate — hỗ trợ PostgreSQL production-grade với sqlx, async migrations, connection pooling. Thay thế SQLite khi cần scale.

📱

Android Agent Platform

Biến điện thoại thành AI Agent chạy 24/7. Foreground Service giữ agent sống, Accessibility Service điều khiển Facebook (đăng bài, bình luận), Messenger (trả lời tự động), Zalo (nhắn tin). Device tools: pin, GPS, notification, clipboard, flashlight. Auto-restart sau reboot.

BẮT ĐẦU NGAY
Chọn cách triển khai phù hợp
Tự cài đặt miễn phí trên thiết bị của bạn, hoặc để chúng tôi lo hạ tầng với gói Cloud.

📥 1. Clone & Build

git clone https://github.com/nguyenduchoai/bizclaw
cd bizclaw && cargo build --release

Hoặc Docker: docker-compose up -d

⚙️ 2. Cấu hình

./target/release/bizclaw init
# Wizard tương tác — chọn provider, nhập API key

🚀 3. Chạy!

./target/release/bizclaw serve
# → Dashboard tại http://localhost:3579

Đã có trợ lý AI — không cần PostgreSQL, Nginx, hay domain.
