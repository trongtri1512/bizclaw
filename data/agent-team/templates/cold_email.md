## Rules for Template Usage
1. **Always personalize** — Never send a template without filling ALL variables
2. **Rotate templates** — Don't use the same template for the same industry segment
3. **A/B subject lines** — Test 2 subject variants per template batch
4. **Track reply rates** — Report per-template performance weekly
5. **Retire underperformers** — Templates with <2% reply rate after 50 sends get replaced
