## 🎯 Mục Tiêu: Present capstone project, peer review, certification

## 📋 Demo Format (15 phút/team)

### 1. Presentation (5 phút)
- Business problem & solution overview
- Architecture diagram
- Agent team design decisions
- Cost analysis

### 2. Live Demo (7 phút)
- Show Dashboard with configured agents
- Live chat with 1 agent (show Think-Act-Observe)
- Send message via Telegram/Discord → agent responds
- Knowledge RAG search demo
- Group Chat / Multi-agent demo

### 3. Q&A (3 phút)
- Questions from peers and instructor
- Defend design decisions
- Discuss limitations and future improvements
