# 📖 Module 21: Capstone — Phân Tích Yêu Cầu & Thiết Kế

> **Phase**: 🎓 CAPSTONE | **Buổi**: 21/24 | **Thời lượng**: 2 giờ

---

## 🎯 Mục Tiêu: Thiết kế hệ thống AI Agent hoàn chỉnh cho doanh nghiệp thực

## 📋 Yêu Cầu Dự Án

### Chọn 1 trong 5 đề tài:

| # | Đề tài | Mô tả |
|---|--------|-------|
| A | **Nhà Hàng AI** | Đặt bàn, menu, feedback, kho nguyên liệu |
| B | **E-Commerce AI** | CSKH, order tracking, product recommendation |
| C | **HR AI Platform** | CV screening, onboarding, employee FAQ |
| D | **Logistics AI** | Tracking, warehouse, delivery optimization |
| E | **Education AI** | Student support, course recommendation, grading |

### Deliverables

1. **Architecture Document** (1 trang A4)
   - Agent team design (roles, providers, tools)
   - Channel strategy
   - Memory strategy
   - Security constraints

2. **System Prompts** (≥ 3 agents)
   - Full 6-section format
   - Few-shot examples
   - Constraints

3. **Knowledge Base** (≥ 5 documents)

4. **Cost Estimation**
   - Monthly cost breakdown by agent
   - Comparison: single vs mixed providers

---

## 📝 Workshop: Requirements Analysis (2 giờ)

### Step 1: Define Users (20 phút)
- WHO uses the system?
- WHAT problems do they have?
- WHEN do they need AI assistance?

### Step 2: Design Agent Team (30 phút)
- How many agents?
- Which provider for each?
- What tools needed?
- Sequential/Parallel/Hierarchical?

### Step 3: Write Architecture Document (40 phút)
- System diagram
- Data flow
- Channel mapping
- Security model

### Step 4: Peer Review (30 phút)
- Present to 1 other team
- Receive feedback
- Iterate design
