# 📖 Module 22: Capstone — Triển Khai Agent System

> **Phase**: 🎓 CAPSTONE | **Buổi**: 22/24 | **Thời lượng**: 2 giờ

---

## 🎯 Mục Tiêu: Implement hệ thống từ thiết kế Module 21

## 📋 Implementation Checklist

### Phase 1: Setup (20 phút)
- [ ] BizClaw running (standalone or platform mode)
- [ ] At least 2 providers configured
- [ ] Ollama installed (optional, for $0 agent)

### Phase 2: Create Agents (30 phút)
- [ ] Create ≥ 3 agents from design document
- [ ] Configure provider per agent
- [ ] Write and test system prompts
- [ ] Select appropriate tools per agent

### Phase 3: Knowledge Base (20 phút)
- [ ] Upload ≥ 5 documents
- [ ] Test search with 10 queries
- [ ] Verify RAG answers quality

### Phase 4: Channel Setup (20 phút)
- [ ] Connect ≥ 1 channel (Telegram recommended)
- [ ] Test end-to-end: external message → agent → response → channel
- [ ] Verify bidirectional communication

### Phase 5: Integration Test (30 phút)
- [ ] Multi-agent broadcast test
- [ ] Group Chat test (if applicable)
- [ ] Failure scenario test (disconnect 1 provider)
- [ ] Quality Gate verification

## 💡 Tips

- Start simple, iterate
- Test EACH component independently before integration
- Document any issues in a bug log
- Take screenshots of Dashboard for presentation
