🏢 Bước 1: Tạo Tenant đầu tiên

Tenant là một AI agent độc lập với cấu hình, model và kênh giao tiếp riêng.

Cách tạo:
1. Vào Tenants → Click "+ New Tenant"
2. Nhập Name (tên hiển thị) và Slug (URL)
3. Chọn Provider (OpenAI, Anthropic, Ollama...)
4. Chọn Model (gpt-4o-mini, claude-3-haiku...)
5. Click Create — Tenant tự động khởi động

🤖 Bước 2: Cấu hình Agent

Mỗi tenant có thể có nhiều agent với vai trò khác nhau.

Cấu hình agent:
1. Vào Tenant Detail → tab Agents
2. Click "+ Thêm Agent"
3. Đặt Name, Role, System Prompt
4. Chọn provider/model riêng cho agent (nếu muốn)
5. Click Tạo — có thể edit lại bất cứ lúc nào

📱 Bước 3: Kết nối kênh

Kết nối agent với Telegram, Zalo, Discord, Email, WhatsApp hoặc Webhook.

Kết nối Telegram:
1. Tạo bot trên @BotFather → lấy Token
2. Vào Tenant Detail → tab Channels
3. Bật Telegram → Paste Bot Token
4. Click 💾 Lưu — Bot sẽ active ngay lập tức!
